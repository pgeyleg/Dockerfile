## DESCRIPTION

Dockerfile can be used to generate Dockerfile  dynamically 


## REQUIREMENTS

*   Ruby 2.4.1+

## INSTALL

```
$ gem install dockerfile
```

or add this line to your Gemfile:

```ruby
gem 'dockerfile'
```
